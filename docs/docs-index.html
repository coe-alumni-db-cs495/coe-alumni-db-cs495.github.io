Documentation

Welcome to the COE Alumni Database documentation. This guide will help you install, use, and extend the application.

Getting Started

The COE Alumni Database is an ASP.NET Core 8 application with a React TypeScript frontend. It uses SQLite for data storage and GitHub Actions for automated deployment.
